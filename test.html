🧪 控制台驗證

在主應用的瀏覽器控制台中運行以下代碼來驗證功能：

// 檢查主題系統
console.log('ThemeManager:', typeof window.ThemeManager);
console.log('當前主題:', window.ThemeManager?.getCurrentTheme()?.id);
console.log('可用主題:', window.ThemeManager?.getAvailableThemes());

// 檢查 Hero Banner 元素
const heroBanner = document.querySelector('[style*="banner"]');
console.log('Hero Banner 元素:', heroBanner ? '✅ 找到' : '❌ 未找到');

// 檢查主標題
const mainTitle = document.querySelector('h1');
console.log('主標題:', mainTitle?.textContent || '未找到');
